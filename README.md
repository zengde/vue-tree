# vue-tree
vue.js tree树组件

[demo](http://zengde.github.io/vue/#/tree)

# props
| 属性	| 类型	| 描述 |
| - | :- | :- | 
|showDetpth	| int	| 最多展开层级 |
|show	| array	| 只展开指定分类中的数据 |


# events
|事件	|参数	|描述|
| - | :- | :- | 
|treeClick	|id |分类id	|叶子点击事件|
